ExtJS-experiments.

These are some experiments I coded while studying ExtJS library.

They are browsable online at http://lbolla.github.com/extjs-experiments/ , but also locally, by doing the following: 

$ git clone https://github.com/lbolla/extjs-experiments.git
$ git checkout -b gh-pages origin/gh-pages

and configuring your webserver to serve extjs-experiments from root.

Ext JS 4 Web Application Development Cookbook

These examples are taken from "Ext JS 4 Web Application Development Cookbook", Stuart Ashworth, Andrew Duncan, Packt Publishing.

They are not "identical" to the examples on the text, but rather experimentations around them.

  • p8 - Creating custom classes using the new Extjs class system
  • p64 - Simple animations
  • p80 - FitLayout
  • p82 - VBox
  • p91 - Accordion
  • p94 - Card
  • p103 - BorderLayout
  • p103a - Stateful Draggable Panels
